# serbian_caps (SerbMode)

Неудобно набирать сербский текст, когда на клавиатуре только русская и английская раскладки?  
Этот скрипт превращает твою обычную клавиатуру в «сербский режим» — без установки сербских раскладок в Windows!

Просто запускаешь скрипт — и можешь набирать сербскую кириллицу и латиницу поверх RU/EN.

## Основные возможности

- Переключение режима одной клавишей: **Caps Lock**
- Индикатор режима: **лампочка Scroll Lock** (горит — режим активен)
- Заглавные буквы — только через **Shift** (Caps Lock на регистр не влияет)
- Работает на **AutoHotkey v2**
- Можно скомпилировать в .exe и добавить в автозагрузку

## Как пользоваться

1. Установи [AutoHotkey v2](https://www.autohotkey.com/)
2. Скачай и запусти `serbian_caps.ahk`
3. Нажми **Caps Lock** — режим включён (лампочка Scroll Lock загорится)
4. Набирай буквы по таблице ниже

## Какие клавиши дают сербские буквы

| Раскладка | Клавиша(и)       | Результат (маленькая) | Результат (заглавная) |
|-----------|------------------|-----------------------|------------------------|
| RU        | j                | љ                     | Љ                      |
| RU        | Л (двойное)      | љ                     | Љ                      |
| RU        | Н (двойное)      | њ                     | Њ                      |
| RU        | Ж (двойное)      | џ                     | Џ                      |
| RU        | Ч (двойное)      | ћ                     | Ћ                      |
| RU        | Д (двойное)      | ђ                     | Ђ                      |
| EN        | C (двойное)      | ć                     | Ć                      |
| EN        | C (тройное)      | č                     | Č                      |
| EN        | S (двойное)      | š                     | Š                      |
| EN        | Z (двойное)      | ž                     | Ž                      |
| EN        | D (двойное)      | đ                     | Đ                      |

### Примеры
- `j` → љ  
- `JJ` → Љ (Shift + j)  
- `cc` → ć  
- `ccc` → č  
- `CC` → Ć (Shift + c)

## Установка и запуск

- Скачай репозиторий
- Запусти `serbian_caps.ahk` (дважды клик)
- Или скомпилируй в .exe через AutoHotkey → «Compile Script» (для удобства автозапуска)
